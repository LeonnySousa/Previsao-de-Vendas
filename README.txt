Este projeto apresenta a solução para o seguinte problema:
Prever as vendas de uma empresa com base nos valores utilizados para anunciar em 3 tipos de midias: TV, Jornal e Rádio;
A base de dados disponível possui as segintes informações:
- Os valores investidos para anunciar na TV, Jornal e Rádio estão em milhares de reais
- As Vendas estão em milhões
